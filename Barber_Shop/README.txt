Функции информационной системы БарберШоп:

    employee.py - Добавление, удаление и вывод в терминал сотрудников.
Вместе с добавлением и удалением сотрудника добавляется и удаляется расписание сотрудника (timetable.py и timetable.csv).
employee.csv - файл с именами сотрудников.
employee_id.csv - файл с id сотрудников. У каждого сотрудника свой id.

    service.py - Добавление, удаление и вывод в терминал услуг.
service.csv - файл с услугами.
service_id.csv - файл с id услуг. У каждой услуги свой id.

    client.py - Добавление, удаление и вывод в терминал записи на приём.
Вместе с добавлением и удалением записи изменяется расписание сотрудника к которому был записан клиент (timetable.py и timetable.csv).
client.csv - файл с записями на приём.
client_id.csv - файл с id записей на приём. У каждой записи на приём свой id.

    timetable.py - Вывод в терминал записей на приём на сегодняшний день.
timetable.csv - файл с доступным временем сотрудников.

    receipt.py - Проведение счёта в системе. В процессе этой функции удаляется запись на приём из расписания и
перемещается в архив записей.
receipt.csv - архивный файл с записями на приём.
receipt_id.csv - файл с id архивных записей на приём.
